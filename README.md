# SQL Assignment Solutions 

Repository containing solutions for 150+ SQL practice questions covering basics to advanced topics.

## What's Inside
This repo has my solutions for the **SQL Assignment** given during CSE coursework. All queries are written and tested in **MySQL**.

**Topics Covered:**
- Basic Queries: `SELECT`, `WHERE`, `ORDER BY`
- Functions: `COUNT`, `SUM`, `AVG`, `MIN`, `MAX`
- Joins: `INNER`, `LEFT`, `RIGHT`, `FULL OUTER`
- Grouping: `GROUP BY`, `HAVING`
- Subqueries & Nested Queries
- DDL & DML: `CREATE`, `INSERT`, `UPDATE`, `DELETE`
- Advanced: `WINDOW FUNCTIONS`, `CTE`, `CASE`

##  Tech Stack
- **Database**: MySQL 8.0+
- **Tools**: MySQL Workbench
